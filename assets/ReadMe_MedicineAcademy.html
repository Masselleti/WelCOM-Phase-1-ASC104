1. System Requirements before you install

  • Windows 10 or Windows 11 (64-bit recommended)
  • At least 2 GB of free disk space
  • Administrator privileges to install applications
  • Internet connection is not required after installation
  • Antivirus or Windows Defender may show a warning for the installer. You can temporarily allow it or disable real-time protection during the installation, and enable it again once the QBank is installed.

2. Installation Steps step by step

Follow these steps to install the offline QBank on your PC:

1
Download the installer file and save it to a trusted folder (for example: Desktop or Downloads). If the file came inside a ZIP archive, extract all files first.
2
Right–click → "Run as administrator" on the installer file, for example:
AMBOSS_Step2_Qbank_2025.exe
3
If Windows SmartScreen shows a blue warning window:
  • Click "More info"
  • Then click "Run anyway"
This is normal behavior for newly installed offline exam software.
4
If your antivirus or Windows Defender blocks the file:
  • Open your security center and choose "Allow" / "Restore" / "Keep anyway" for the installer.
  • You may temporarily turn off real-time protection only during the installation,
  • After installation is finished, turn your antivirus protection back on and keep the installed folder in the allow list.
5
Follow the installation wizard:
  • Choose your installation folder (default is recommended)
  • Optionally create a Desktop shortcut
  • Click "Install" and wait for the process to complete
6
Click "Finish" to close the installer. You can choose to launch the QBank immediately after installation.

3. Running the QBank after installation

After installation, you can open the QBank in any of the following ways:

  • Use the Desktop shortcut created by the installer (for example: "Medicine Academy – QBank")
  • Open the Start Menu and search for "Medicine Academy"

When launched, the application will automatically load all required HTML, JS, and data files from the installed Data and js folders. No manual configuration is needed.

4. Troubleshooting common issues

Issue A – The program does not open at all

  • Make sure the installation completed successfully without errors.
  • Open the installation folder (for example: C:\Program Files\MedicineAcademy\...) and verify that you see:
    • The main QBank executable
    • A Data folder
    • A js folder (or assets folder for some engines)
    • index.html file
  • If any of these are missing, reinstall the QBank using the installer.

Issue B – Windows Defender or antivirus blocks the app

  • This QBank uses a local HTML/JS engine and database, which can sometimes trigger false–positive alerts.
  • Open your antivirus or Windows Security → find the blocked file → choose "Allow" / "Restore" / "Allow on this device".
  • You may temporarily disable real-time protection while running the installer, then enable it again after installation.
  • Add the installation folder to the Exclusions / Allow list if necessary.

Issue C – Blank window or content not loading

  • Close the application completely and run it again.
  • Ensure that antivirus software has not removed any files from the Data or js/assets folders.
  • Reinstall the QBank if required.
Still having issues?
Please contact Medicine Academy Support and include:
  • Your Windows version (e.g. Windows 11 Pro 64-bit)
  • A screenshot of any error message
  • A screenshot of the installation folder

5. Android Version (if included) APK installation

If your package contains an Android version (APK file), you can install it on your phone or tablet as follows:

  • Copy the APK file (for example: AMBOSS_Step2_Qbank_2025.apk) to your Android device.
  • On your phone, open Settings → Security or Apps & notifications.
  • Allow installation from "Unknown sources" or enable "Install unknown apps" for the app you use to open the file (for example: Chrome, Files, or your file manager).
  • Open the APK file and tap Install.
  • If Google Play Protect shows a warning, tap "More details" then "Install anyway", provided you downloaded the file directly from Medicine Academy.

Once installed, the Android QBank runs fully offline. You can later disable Install unknown apps again if you prefer.
